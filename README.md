# prework-study-guide

## Description

This is a study guide for UCLA Extension Web Development Bootcamp students to learn the basics of html, css, js, and git.

## Installation

N/A

## Usage

To use the Study Guide, review the notes in each section. Open Chrome DevTools by pressing `Command+Option+I` (MacOS) or `Control+Shift+I` (Windows). A console panel will open at the bottom or side of the webpage in the browser. There will be a list of topics that can be learned as well as a suggestion for which topic to study first.
